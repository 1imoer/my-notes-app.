# ما الذي أفعله الآن؟

## المسار 1: PWA
- أضف الملفات الموجودة في مجلد `pwa/` إلى جذر الموقع.
- أضف أسطر الرابط داخل `<head>` في `index.html`.
- انشر على GitHub Pages.
- افتح الموقع من الهاتف ثم اختر التثبيت.

## المسار 2: APK
- ارفع مجلد `apk-wrapper/` إلى GitHub كمستودع جديد.
- فعّل Actions.
- انتظر إنشاء APK.
- حمّل الملف وثبّته على الهاتف.
